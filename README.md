Política de Privacidad — FocusLoops

Última actualización: 2025-01-01

FocusLoops respeta la privacidad de los usuarios. Esta aplicación no recopila, almacena ni comparte ningún dato personal.

1. Información que recopilamos

La aplicación no recopila ningún tipo de información personal, sensible o identificable del usuario.

FocusLoops no solicita:

Nombre

Email

Ubicación

Contactos

Archivos

Fotos

Datos financieros

Identificadores del dispositivo

Información de uso

2. Información que comparte la aplicación

FocusLoops no comparte ningún tipo de dato con terceros, empresas, servidores ni servicios externos.

3. Compras dentro de la aplicación

La aplicación ofrece una compra única para desbloquear funciones premium.
La compra es gestionada directamente por Google Play Billing, y FocusLoops no recibe ningún dato personal del usuario.

Google Play puede procesar datos relacionados con la compra según sus propios términos.
Consulta la política de privacidad de Google para más detalles:

https://policies.google.com/privacy
4. Conexión a internet

La aplicación no requiere conexión a internet para funcionar.
Solo se conecta a Google Play para validar compras.

5. Seguridad

Como la aplicación no gestiona datos personales, no almacena ni transmite información privada.

6. Cambios en esta política

Nos reservamos el derecho de actualizar esta política. Cualquier cambio será reflejado en esta misma página.
